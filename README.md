# Project 9 at Origin Code Academy
## Movie Finder using OMBD
### Technologies Used:
- HTML
- CSS
    + Vital.css Framework
- Javascript
    + Angular
    + Angular-UI
- Gulp

After cloning run
```
npm install
```

and then
```
gulp serve
```

to get it working on localhost. Defaults to port 8889, which is configurable in the gulpfile in the 'connect' task.
